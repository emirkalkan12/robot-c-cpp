\doxysubsection{Detailed Description}
Manages safe movement of the robot using the IR sensor and robot controller. 

The \doxylink{class_safe_navigation}{Safe\+Navigation} class interacts with the \doxylink{class_i_r_sensor}{IRSensor} to retrieve distance data and with the \doxylink{class_robot_controler}{Robot\+Controler} to control the robot\textquotesingle{}s movements. It ensures that the robot moves safely by considering sensor readings during forward and backward motion.
