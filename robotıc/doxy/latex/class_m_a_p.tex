\doxysubsection{Detailed Description}
Represents a map system responsible for storing, updating, and managing the map. 

The \doxylink{class_m_a_p}{MAP} class provides functionalities to manage a grid-\/based map. It includes methods to insert points, modify and access grid values, clear the map, print its details, and display the map visually.
