\doxysubsection{Detailed Description}
Maps the environment using lidar sensors and robot control data. 

The \doxylink{class_mapper}{Mapper} class manages a grid-\/based map using data from a \doxylink{class_lidar_sensor}{Lidar\+Sensor} and a \doxylink{class_robot_controler}{Robot\+Controler}. It provides methods to update the map based on sensor data, save the map to a file, and display the map visually.
