\doxysubsection{Detailed Description}
\doxylink{class_robot_controler}{Robot\+Controler} class. 

Controls the movement of a robot in a 2D space.

The \doxylink{class_robot_controler}{Robot\+Controler} class provides functionalities for controlling the movement of a robot in a 2D space. It uses the \doxylink{class_festo_robot_a_p_i}{Festo\+Robot\+API} class to communicate with the robot and send commands for moving the robot in different directions.
