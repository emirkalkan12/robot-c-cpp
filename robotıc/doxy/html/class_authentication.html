bool Authentication::authenticate ( )

Authenticates the user based on name, surname, and access code.

This function prompts the user to enter their name, surname, and access code. It then checks the entered access code against the encrypted access code stored in the robot operator. If the access code is correct, the user is authenticated, otherwise, they are denied access.

Returns
Returns true if the user is successfully authenticated, false otherwise.
