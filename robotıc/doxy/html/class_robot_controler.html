Detailed Description

RobotControler class.

Controls the movement of a robot in a 2D space.

The RobotControler class provides functionalities for controlling the movement of a robot in a 2D space. It uses the FestoRobotAPI class to communicate with the robot and send commands for moving the robot in different directions.
